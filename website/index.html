About

Built for manga lovers on desktop.

Inkflow is a lightweight desktop manga reader focused on reading comfort, fast downloads, and a polished app experience. It's built with performance in mind, no subscription, no tracking, no ads.

The project is open source and community-driven. Contributions, bug reports, and feature requests are always welcome on GitHub.
